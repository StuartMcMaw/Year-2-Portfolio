Postr - IxD Project

Tasked to create a new product to display information, I had the idea to create a 'Digital Poster' that will link up with an App that allows what is shown to be changed at any time.

I then began to think how to develop this idea even further, and thought that I could make certain posters more 'Limited' than others by only making some available to purchase at a certain time and/or available in limited quantities. By doing so this would allow certain posters to become highly valuable and sought after.
